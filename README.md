## Quick Start

**Prerequisites:** JDK 8+, MySQL, Maven

**Setup:**
```bash
mysql -u root -p < setup.sql
mvn clean install
./run-sms.sh        # Linux/Mac
run-sms.bat         # Windows
```

**Login:** admin / 123456
